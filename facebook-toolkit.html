Facebook Toolkit

เร็ว ๆ นี้ กำลังพัฒนา • สำหรับ facebook.com

เครื่องมือจัดการ Facebook อัตโนมัติ — follow/unfollow เพจและคนตามเงื่อนไข ไลก์โพสต์ในฟีดตามเกณฑ์ ตั้งหน่วงเวลาแบบสุ่มเหมือนคนจริง และคุมความเสี่ยงด้วยลิมิตต่อชั่วโมง/ต่อวัน ทำงานต่อแม้ปิด popup

🔔 แจ้งเตือนเมื่อพร้อม ดูตัวอย่าง

ความสามารถ

ออกแบบสำหรับแอดมินเพจและนักการตลาดที่อยากโตบน Facebook โดยไม่ต้องนั่งกดเอง

➕

Follow อัตโนมัติตามเงื่อนไข

เปิดหน้าผู้ติดตามเพจอื่นหรือผลค้นหา แล้วกด Start ระบบกดติดตามให้ พร้อม Whitelist

🧹

Unfollow คนที่ไม่ Follow กลับ

เก็บกวาดรายการติดตามให้สะอาด เลือกเฉพาะคนที่ไม่ติดตามกลับ

❤️

ไลก์โพสต์ตามเกณฑ์

ไลก์เฉพาะโพสต์ในฟีดที่ผ่านคำค้น ข้ามโพสต์โฆษณา และเลื่อนฟีดต่อให้อัตโนมัติ

⏱️

หน่วงเวลาแบบสุ่ม

ตั้งช่วงหน่วงเวลา min–max วินาที ระบบสุ่มจังหวะให้เหมือนคนจริง ลดความเสี่ยงโดนตรวจจับ

📊

ลิมิตต่อชั่วโมง / ต่อวัน

กำหนดเพดาน action พร้อมตัวนับ usage แบบเรียลไทม์ และหยุดอัตโนมัติเมื่อถึงลิมิต

🔌

ทำงานต่อแม้ปิด popup

เครื่องยนต์รันเบื้องหลังด้วย service worker ปิดหน้าต่างได้ งานยังเดินต่อ

ตัวอย่างหน้าตา (ออกแบบไว้)

ภาพตัวอย่างดีไซน์ก่อนเปิดตัวจริง — หน้าตาจริงอาจปรับเปลี่ยนได้

วิธีใช้งาน (แผนการออกแบบ)

เปิดหน้า Facebook แล้วกด Start ได้เลย

①

เลือกโหมด Follow หรือ Like

เปิด popup เลือกแท็บที่ต้องการ ตั้งกลุ่มเป้าหมายและเงื่อนไข

②

ตั้งลิมิตและหน่วงเวลา

กำหนดจำนวนสูงสุดต่อรอบ ช่วงหน่วงเวลา และ Whitelist ที่ห้ามแตะ

③

กด Start

ระบบทำงานตามเงื่อนไขให้อัตโนมัติ พร้อมตัวนับแสดงความคืบหน้า

④

คุมความเสี่ยงในแท็บตั้งค่า

ดูตัวนับ usage รายชั่วโมง/รายวัน และปรับลิมิตได้ทุกเมื่อ

💡 เริ่มต้นด้วยลิมิตต่ำ ๆ และหน่วงเวลาช้า ๆ ก่อน แล้วค่อยปรับขึ้นทีละนิด เพื่อความปลอดภัยของบัญชี
🔔 กำลังพัฒนา

Facebook Toolkit กำลังจะมา เร็ว ๆ นี้

ส่วนขยายนี้อยู่ระหว่างพัฒนา กดติดตาม Facebook Page ของ GCK เพื่อรับแจ้งเตือนทันทีที่เปิดให้ติดตั้งบน Chrome Web Store

💬 ติดตามเพื่อรับแจ้งเตือน ✉️ อีเมล

สิทธิ์การเข้าถึงที่วางแผนจะใช้

  • storage — บันทึกการตั้งค่าและตัวนับลิมิตในเครื่อง
  • tabs, scripting — ตรวจสอบแท็บ Facebook ที่เปิดอยู่และสั่งงานบนหน้าเว็บ
  • host: www.facebook.com — อ่านและคลิกองค์ประกอบที่ปรากฏบนหน้าจอเหมือนผู้ใช้ทั่วไป
🔒 ส่วนขยายของ GCK ยึดหลัก ไม่ส่งข้อมูลออกนอกเครื่อง การตั้งค่าทั้งหมดเก็บใน chrome.storage.local และลบออกได้ตลอดเวลา อ่านนโยบายฉบับเต็มได้ที่ หน้านโยบายความเป็นส่วนตัว
⚠️ Facebook มีระบบตรวจจับการใช้งานอัตโนมัติ — โปรดใช้ด้วยความระมัดระวัง ตั้งหน่วงเวลาช้า ๆ และลิมิตต่ำ ๆ การใช้เครื่องมืออัตโนมัติเป็นความรับผิดชอบของผู้ใช้เอง

ส่วนขยายนี้เป็นเครื่องมือของบุคคลที่สาม ไม่ได้เกี่ยวข้องหรือได้รับการรับรองจาก Facebook / Meta
