Twitter Video Grabber

พร้อมใช้งาน เวอร์ชัน 4.0.0 • สำหรับ Twitter / X

ดาวน์โหลดวิดีโอจาก Twitter / X ได้ในคลิกเดียว — เปิดทวีตที่มีวิดีโอแล้วกดเล่น หรือวาง URL ทวีตตรง ๆ ส่วนขยายจะดักจับลิงก์วิดีโอให้อัตโนมัติ เลือกความละเอียดที่ต้องการ ดาวน์โหลดเป็น MP4 และ export รายการเป็น CSV ได้

＋ เพิ่มลงใน Chrome ดูตัวอย่าง

ความสามารถ

เครื่องมือเซฟวิดีโอ Twitter/X ที่เร็ว ง่าย และทำงานในเครื่องคุณล้วน ๆ

🎬

ดักจับวิดีโออัตโนมัติ

ส่วนขยายฟัง network request ของ video.twimg.com แล้วเก็บลิงก์วิดีโอที่เจอให้อัตโนมัติ แยกตามแท็บ

🔗

วาง URL ทวีตได้ตรง ๆ

ไม่ต้องเปิดเล่นวิดีโอก็ได้ — วางลิงก์ทวีต (twitter.com หรือ x.com) แล้วกดค้นหาเพื่อดึงวิดีโอออกมา

🎚️

เลือกความละเอียด

แต่ละวิดีโอมักมีหลายขนาด (320p / 360p / 720p) เลือกตัวที่คมที่สุดได้ตามต้องการ

⬇️

ดาวน์โหลด MP4 ปุ่มเดียว

ไฟล์ .mp4 ดาวน์โหลดตรงได้ทันที ส่วน HLS .m3u8 คัดลอกลิงก์ไปใช้กับ ffmpeg ได้

🔢

Badge นับจำนวน

ตัวเลขบนไอคอนบอกจำนวนวิดีโอที่เจอในแท็บปัจจุบัน รู้ทันทีว่ามีอะไรให้เซฟบ้าง

📋

Export เป็น CSV

รวมทุกลิงก์ที่เจอในแท็บ (ชนิด, ความละเอียด, ลิงก์วิดีโอ, ลิงก์ทวีต, เวลา) เป็นไฟล์ CSV เปิดใน Excel/Sheets ได้

ตัวอย่างการใช้งาน

ดูหน้าตาจริงก่อนติดตั้ง

วิธีใช้งาน

เปิดทวีตที่มีวิดีโอ หรือวางลิงก์ตรง ๆ ก็ได้

①

เปิดทวีตแล้วกดเล่นวิดีโอ

เปิดทวีตที่มีวิดีโอ กดเล่นสัก 1–2 วินาที เพื่อให้เบราว์เซอร์เริ่มโหลดวิดีโอ ตัวเลขสีฟ้าบนไอคอนจะบอกจำนวนวิดีโอที่เจอ

②

เลือกความละเอียดและดาวน์โหลด

คลิกไอคอนเพื่อเปิด popup เลือกความละเอียดที่ต้องการ (เลือกตัวใหญ่สุดถ้าต้องการคุณภาพดีที่สุด) แล้วกด ดาวน์โหลด

③

หรือวาง URL ทวีต / Export CSV

ไม่อยากเปิดเล่นก็วาง URL ทวเตในช่องค้นหาได้เลย และถ้าต้องการลิสต์ทั้งหมด กด Export CSV เพื่อบันทึกทุกลิงก์ในแท็บ

💡 วิดีโอแบบ HLS (.m3u8) ถูกหั่นเป็นหลายชิ้น ดาวน์โหลดไฟล์เดียวไม่ได้วิดีโอเต็ม — ให้คัดลอกลิงก์ไปรวมด้วย ffmpeg -i "ลิงก์ m3u8" -c copy output.mp4

วิธีติดตั้ง

ติดตั้งผ่าน Chrome Web Store ได้ในไม่กี่คลิก

1

กด "เพิ่มลงใน Chrome"

คลิกปุ่ม เพิ่มลงใน Chrome บนหน้านี้ หรือค้นหา "Twitter Video Grabber" ใน Chrome Web Store โดยตรง

2

ยืนยันการติดตั้ง

กด เพิ่มส่วนขยาย ในหน้าต่างยืนยัน Chrome จะติดตั้งให้อัตโนมัติภายในไม่กี่วินาที

3

ปักหมุดไอคอน

คลิกไอคอนต่อจิ๊กซอว์ 🧩 มุมขวาบน แล้วกดหมุด 📌 ข้างชื่อ Twitter Video Grabber

4

เปิดทวีตที่มีวิดีโอ

ไปยังทวีตที่มีวิดีโอบน Twitter / X กดเล่น แล้วคลิกไอคอนเพื่อเลือกและดาวน์โหลด

สิทธิ์การเข้าถึงที่ใช้

  • webRequest, declarativeNetRequest — ดักจับลิงก์วิดีโอที่หน้าเว็บโหลดจาก video.twimg.com
  • downloads — ดาวน์โหลดไฟล์วิดีโอเมื่อผู้ใช้กดปุ่ม
  • storage — เก็บรายการวิดีโอที่เจอแยกตามแท็บ
  • tabs — รู้ว่าวิดีโอที่เจออยู่ในแท็บไหน เพื่อแสดง badge และจัดรายการให้ถูกต้อง
  • เข้าถึงเฉพาะโดเมน twitter.com, x.com, video.twimg.com, api.twitter.com
🔒 ส่วนขยายนี้ ทำงานในเครื่องของคุณล้วน ๆ ไม่ส่งข้อมูลออกนอกเครื่อง ลิงก์วิดีโอที่เก็บอยู่ใน chrome.storage และล้างได้ตลอดเวลา อ่านนโยบายฉบับเต็มได้ที่ หน้านโยบายความเป็นส่วนตัว

โปรดใช้เพื่อการส่วนตัวเท่านั้น และเคารพลิขสิทธิ์ของเจ้าของเนื้อหาและข้อกำหนดของแพลตฟอร์ม ส่วนขยายนี้เป็นเครื่องมือของบุคคลที่สาม ไม่ได้เกี่ยวข้องหรือได้รับการรับรองจาก Twitter / X
