YouTube Toolkit

เร็ว ๆ นี้ กำลังพัฒนา • สำหรับ youtube.com

เครื่องมือจัดการ YouTube อัตโนมัติ — subscribe/unsubscribe ช่องตามเงื่อนไข ไลก์วิดีโอที่ผ่านเกณฑ์ยอดวิว/คำค้น กรองตามยอดผู้ติดตาม ตั้งหน่วงเวลาแบบสุ่ม และคุมความเสี่ยงด้วยลิมิตต่อชั่วโมง/ต่อวัน

🔔 แจ้งเตือนเมื่อพร้อม ดูตัวอย่าง

ความสามารถ

ออกแบบสำหรับครีเอเตอร์และนักการตลาดที่อยากเติบโตบน YouTube อย่างเป็นระบบ

➕

Subscribe ตามเงื่อนไข

subscribe ช่องจากผลค้นหา กรองตามยอดผู้ติดตามขั้นต่ำ พร้อม Whitelist

🧹

Unsubscribe เก็บกวาด

ยกเลิกการติดตามช่องที่ไม่ต้องการได้เป็นชุด เลือกเงื่อนไขได้

❤️

ไลก์วิดีโอตามเกณฑ์

ไลก์เฉพาะวิดีโอที่ผ่านยอดวิวขั้นต่ำ ความยาว และคำค้นในชื่อวิดีโอ

🔎

กรองละเอียด

กรองตามยอดวิว ยอดผู้ติดตาม และ keyword เจาะเฉพาะกลุ่มเป้าหมายจริง

📊

ลิมิตต่อชั่วโมง / ต่อวัน

กำหนดเพดาน action พร้อมตัวนับเรียลไทม์ หยุดอัตโนมัติเมื่อถึงลิมิต

🔌

ทำงานต่อแม้ปิด popup

เครื่องยนต์รันเบื้องหลัง ปิดหน้าต่าง popup ได้ งานยังเดินต่อ

ตัวอย่างหน้าตา (ออกแบบไว้)

ภาพตัวอย่างดีไซน์ก่อนเปิดตัวจริง — หน้าตาจริงอาจปรับเปลี่ยนได้

วิธีใช้งาน (แผนการออกแบบ)

เปิดหน้า YouTube แล้วกด Start ได้เลย

①

เลือกโหมด Subscribe หรือ Like

เปิด popup เลือกแท็บ ตั้งคำค้นและเกณฑ์ที่ต้องการ

②

ตั้งตัวกรอง

กำหนดยอดผู้ติดตาม/ยอดวิวขั้นต่ำ และ keyword ในชื่อ

③

กด Start

ระบบ subscribe หรือไลก์เฉพาะรายการที่ผ่านเกณฑ์ให้อัตโนมัติ

④

คุมลิมิตในแท็บตั้งค่า

ดูตัวนับ usage และปรับเพดานต่อชั่วโมง/ต่อวันได้ทุกเมื่อ

💡 ตั้งเกณฑ์ยอดวิว/ผู้ติดตามให้ตรงกลุ่มเป้าหมาย เพื่อให้การติดตามมีคุณภาพมากกว่าปริมาณ
🔔 กำลังพัฒนา

YouTube Toolkit กำลังจะมา เร็ว ๆ นี้

ส่วนขยายนี้อยู่ระหว่างพัฒนา กดติดตาม Facebook Page ของ GCK เพื่อรับแจ้งเตือนทันทีที่เปิดให้ติดตั้งบน Chrome Web Store

💬 ติดตามเพื่อรับแจ้งเตือน ✉️ อีเมล

สิทธิ์การเข้าถึงที่วางแผนจะใช้

  • storage — บันทึกการตั้งค่าและตัวนับลิมิตในเครื่อง
  • tabs, scripting — ตรวจสอบแท็บ YouTube ที่เปิดอยู่และสั่งงานบนหน้าเว็บ
  • host: www.youtube.com — อ่านและคลิกองค์ประกอบที่ปรากฏบนหน้าจอเหมือนผู้ใช้ทั่วไป
🔒 ส่วนขยายของ GCK ยึดหลัก ไม่ส่งข้อมูลออกนอกเครื่อง การตั้งค่าทั้งหมดเก็บใน chrome.storage.local และลบออกได้ตลอดเวลา อ่านนโยบายฉบับเต็มได้ที่ หน้านโยบายความเป็นส่วนตัว
⚠️ YouTube มีระบบตรวจจับการใช้งานอัตโนมัติ — โปรดใช้ด้วยความระมัดระวัง ตั้งหน่วงเวลาช้า ๆ และลิมิตต่ำ ๆ การใช้เครื่องมืออัตโนมัติเป็นความรับผิดชอบของผู้ใช้เอง

ส่วนขยายนี้เป็นเครื่องมือของบุคคลที่สาม ไม่ได้เกี่ยวข้องหรือได้รับการรับรองจาก YouTube / Google
