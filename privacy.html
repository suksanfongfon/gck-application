นโยบายความเป็นส่วนตัว

ปรับปรุงล่าสุด: 2 มิถุนายน 2026

นโยบายความเป็นส่วนตัวฉบับนี้ครอบคลุมส่วนขยาย Chrome ทุกตัวที่พัฒนาและเผยแพร่ภายใต้ชื่อ GCK Chrome Extension ได้แก่ X Toolkit, Lazada Product Finder และส่วนขยายอื่น ๆ ที่จะเพิ่มในอนาคต (เรียกรวมว่า "ส่วนขยาย")

เราให้ความสำคัญกับความเป็นส่วนตัวของผู้ใช้เป็นอย่างสูง นโยบายนี้อธิบายอย่างชัดเจนว่าเราเก็บข้อมูลใดบ้าง ใช้อย่างไร และคุณมีสิทธิ์อะไร

1. หลักการทั่วไป

ส่วนขยายทุกตัวของ GCK Chrome Extension ยึดหลักการดังนี้

  • ทำงานในเครื่องของผู้ใช้ (locally) เป็นหลัก
  • ไม่เก็บ ไม่ส่ง และไม่ขายข้อมูลส่วนบุคคลให้บุคคลที่สาม
  • ไม่ใช้ข้อมูลเพื่อโฆษณาหรือวิเคราะห์พฤติกรรมผู้ใช้
  • ขอสิทธิ์เฉพาะที่จำเป็นต่อการทำงานของฟีเจอร์จริงเท่านั้น
  • ไม่ใช้ Remote Code — ทุก script เป็นไฟล์ local ใน package

2. ข้อมูลที่เก็บรวบรวม

2.1 ข้อมูลที่เก็บในเครื่องผู้ใช้ (Local Storage)

ส่วนขยายทุกตัวใช้ chrome.storage.local เพื่อบันทึกข้อมูลต่อไปนี้ ในเครื่องของผู้ใช้เท่านั้น:

  • การตั้งค่าที่ผู้ใช้กำหนดเอง เช่น ความถี่ ลิมิต ตัวกรอง
  • ตัวนับการทำงาน (rate limit counter) สำหรับคำนวณช่วงพัก
  • Whitelist หรือรายการที่ผู้ใช้กรอกเอง
  • การตั้งค่า API key / affiliate config ที่ผู้ใช้กรอกเอง

ข้อมูลทั้งหมดนี้ ไม่ถูกส่งออกสู่เซิร์ฟเวอร์ใด ๆ และจะถูกลบทิ้งเมื่อผู้ใช้ถอนการติดตั้งส่วนขยาย

2.2 ข้อมูลที่ส่วนขยายไม่เก็บ

  • ชื่อ อีเมล หรือข้อมูลบัญชีผู้ใช้
  • ประวัติการเข้าชมเว็บไซต์
  • เนื้อหาของโพสต์หรือข้อความส่วนตัว
  • ข้อมูลตำแหน่งที่ตั้ง
  • ข้อมูลการชำระเงินหรือบัตรเครดิต

3. สิทธิ์การเข้าถึงและเหตุผล

ด้านล่างนี้คือสิทธิ์ที่ส่วนขยายแต่ละตัวใช้ พร้อมเหตุผลที่จำเป็นต้องใช้สิทธิ์นั้น

4. Remote Code

ส่วนขยายทุกตัวของ GCK Chrome Extension ไม่ใช้ Remote Code ในทุกกรณี ทุก script (content.js, background.js, popup.js) เป็นไฟล์ local ที่บรรจุอยู่ใน extension package เท่านั้น ไม่มีการโหลด script จาก URL ภายนอก ไม่มีการใช้ eval() หรือเทียบเท่า

5. การส่งข้อมูลออกภายนอก

ส่วนขยายส่งข้อมูลออกสู่เซิร์ฟเวอร์ภายนอกเฉพาะเมื่อผู้ใช้ เริ่มต้นการกระทำนั้นเองโดยตรง เช่น:

  • X Toolkit: ส่ง request ไปยัง x.com / twitter.com เมื่อผู้ใช้กด Start เพื่อ follow/unfollow/repost ตามปกติของแพลตฟอร์ม และส่ง request ไปยัง Lazada API เฉพาะเมื่อผู้ใช้เปิดโหมด Lazada และกรอก API key ของตัวเอง
  • Lazada Product Finder: เปิดหน้า lazada.co.th ตาม keyword ที่ผู้ใช้กรอก เพื่อดึงข้อมูลสินค้าที่เป็นสาธารณะ

เราไม่ส่งข้อมูลใด ๆ ไปยังเซิร์ฟเวอร์ของ GCK Chrome Extension เราไม่มีเซิร์ฟเวอร์ backend สำหรับเก็บข้อมูลผู้ใช้

6. การแบ่งปันข้อมูลกับบุคคลที่สาม

เราไม่ขาย ไม่เช่า และไม่แบ่งปันข้อมูลของผู้ใช้ ให้กับบุคคลที่สามในทุกกรณี ส่วนขยายไม่ฝัง SDK วิเคราะห์ข้อมูล (analytics) หรือโฆษณาใด ๆ

7. ความปลอดภัยของข้อมูล

ข้อมูลทั้งหมดที่บันทึกในเครื่องถูกจัดเก็บผ่าน chrome.storage.local ซึ่งได้รับการปกป้องโดย sandbox ของ Chrome และระบบปฏิบัติการ ความปลอดภัยของข้อมูล API key ที่ผู้ใช้กรอกเองขึ้นอยู่กับ การรักษาความปลอดภัยของอุปกรณ์ของผู้ใช้ด้วย

8. สิทธิ์ของผู้ใช้

คุณสามารถ:

  • ลบข้อมูลทั้งหมด — ถอนการติดตั้งส่วนขยาย Chrome จะลบข้อมูลใน chrome.storage.local ให้อัตโนมัติ
  • ล้างข้อมูลเฉพาะส่วน — ไปที่ chrome://extensions → คลิกรายละเอียดส่วนขยาย → กด "ล้างข้อมูล"
  • ตรวจสอบข้อมูล — เปิด DevTools → Application → Storage → Extension Storage เพื่อดูข้อมูลที่บันทึกไว้
  • ติดต่อเรา — หากมีคำถามหรือต้องการความช่วยเหลือ ส่งอีเมลมาที่ suksan.cmu@gmail.com

9. ผู้เยาว์

ส่วนขยายของเราไม่ได้ออกแบบมาสำหรับผู้ที่มีอายุต่ำกว่า 13 ปี เราไม่ทราบและไม่ประสงค์จะเก็บข้อมูลจากผู้เยาว์โดยเจตนา

10. การเปลี่ยนแปลงนโยบาย

เราอาจปรับปรุงนโยบายนี้เมื่อมีการเพิ่มส่วนขยายใหม่หรือเปลี่ยนแปลงฟีเจอร์ การเปลี่ยนแปลงจะมีผลเมื่อเผยแพร่บนหน้านี้ โปรดตรวจสอบวันที่ "ปรับปรุงล่าสุด" ด้านบนเป็นประจำ
