TikTok Toolkit

พร้อมใช้งาน เวอร์ชัน 1.0.0 • สำหรับ tiktok.com

เครื่องมือจัดการบัญชี TikTok อัตโนมัติ — follow/unfollow ตามเงื่อนไข ไลก์วิดีโอในฟีดตามเกณฑ์ยอดวิว/คำค้น ตั้งหน่วงเวลาแบบสุ่มเหมือนคนจริง และคุมความเสี่ยงด้วยลิมิตต่อชั่วโมง/ต่อวัน ทำงานต่อแม้ปิด popup

＋ เพิ่มลงใน Chrome ดูตัวอย่าง

ความสามารถ

ออกแบบสำหรับครีเอเตอร์และนักการตลาดที่อยากโตบน TikTok โดยไม่ต้องนั่งกดเอง

➕

Follow อัตโนมัติตามเงื่อนไข

เปิดหน้าค้นหาคนหรือ follower ของบัญชีอื่น แล้วกด Start — ระบบกดติดตามให้ พร้อม Whitelist กันบัญชีที่ไม่ต้องการ

🧹

Unfollow คนที่ไม่ Follow กลับ

เก็บกวาด Following ให้สะอาด เลือกเฉพาะ Non-mutual (คนที่ไม่ติดตามกลับ) และตั้ง Whitelist รายชื่อที่ห้าม unfollow ได้

❤️

ไลก์วิดีโอตามเกณฑ์

ไลก์เฉพาะวิดีโอในฟีดที่ผ่านเกณฑ์ — ยอดวิวขั้นต่ำ, ยอดไลก์ขั้นต่ำ และคำค้นใน caption เลื่อนฟีดต่อให้อัตโนมัติ

⏱️

หน่วงเวลาแบบสุ่ม

ตั้งช่วงหน่วงเวลา min–max วินาที ระบบสุ่มจังหวะให้เหมือนคนจริง ลดความเสี่ยงโดนระบบตรวจจับบอท

📊

ลิมิตต่อชั่วโมง / ต่อวัน

กำหนดเพดานจำนวน action ต่อชั่วโมงและต่อวัน มีตัวนับ usage แบบเรียลไทม์ พร้อมหยุดอัตโนมัติเมื่อถึงลิมิต

🔌

ทำงานต่อแม้ปิด popup

เครื่องยนต์รันอยู่เบื้องหลัง (service worker) ปิดหน้าต่าง popup ได้ งานยังเดินต่อ และกด Stop หยุดได้ทันที

ตัวอย่างการใช้งาน

ดูหน้าตาจริงก่อนติดตั้ง

วิธีใช้งาน

4 แท็บ ใช้งานง่าย เปิดหน้า TikTok แล้วกด Start ได้เลย

①

แท็บ "Follow" — กดติดตามอัตโนมัติ

เปิดหน้า ค้นหาคน หรือหน้า follower ของบัญชีอื่น บน TikTok ตั้งจำนวนสูงสุด/รอบ และช่วงหน่วงเวลา ใส่ Whitelist ได้ถ้าต้องการ แล้วกด Start Follow

②

แท็บ "Unfollow" — เก็บกวาด Following

เปิดหน้า Following ของบัญชีตัวเอง เปิดสวิตช์ "เฉพาะคนที่ไม่ Follow กลับ" ใส่ Whitelist ชื่อที่ห้าม unfollow แล้วกด Start Unfollow

③

แท็บ "Like" — ไลก์ฟีดตามเกณฑ์

เปิดหน้า For You หรือ Following ตั้งยอดวิว/ยอดไลก์ขั้นต่ำ และคำค้นใน caption แล้วกด Start Like ระบบจะเลื่อนฟีดและไลก์เฉพาะวิดีโอที่ผ่านเกณฑ์

④

แท็บ "ตั้งค่า" — คุมความถี่และลิมิต

กำหนดลิมิตต่อชั่วโมงและต่อวัน ดูตัวนับ usage แบบเรียลไทม์ และรีเซ็ตตัวนับลิมิตได้ทุกเมื่อ

💡 เริ่มต้นด้วยลิมิตต่ำ ๆ และหน่วงเวลาช้า ๆ ก่อน แล้วค่อยปรับขึ้นทีละนิด เพื่อความปลอดภัยของบัญชีคุณ

วิธีติดตั้ง

ติดตั้งผ่าน Chrome Web Store ได้ในไม่กี่คลิก

1

กด "เพิ่มลงใน Chrome"

คลิกปุ่ม เพิ่มลงใน Chrome บนหน้านี้ หรือค้นหา "TikTok Toolkit" ใน Chrome Web Store โดยตรง

2

ยืนยันการติดตั้ง

กด เพิ่มส่วนขยาย ในหน้าต่างยืนยัน Chrome จะติดตั้งให้อัตโนมัติภายในไม่กี่วินาที

3

ปักหมุดไอคอน

คลิกไอคอนต่อจิ๊กซอว์ 🧩 มุมขวาบน แล้วกดหมุด 📌 ข้างชื่อ TikTok Toolkit

4

เปิดหน้า TikTok และเริ่มใช้งาน

เปิด tiktok.com ไปยังหน้าที่ต้องการ (ค้นหาคน / Following / For You) คลิกไอคอนเพื่อเปิด popup เลือกโหมด แล้วกด Start

สิทธิ์การเข้าถึงที่ใช้

  • storage — บันทึกการตั้งค่าและ ตัวนับลิมิต (รายชั่วโมง/รายวัน) ในเครื่อง
  • tabs, scripting — ตรวจสอบแท็บ TikTok ที่เปิดอยู่และสั่งงานบนหน้าเว็บ
  • เข้าถึง www.tiktok.com เท่านั้น — อ่านและคลิกองค์ประกอบที่ปรากฏบนหน้าจอเหมือนผู้ใช้ทั่วไป
🔒 ส่วนขยายนี้ ไม่ส่งข้อมูลออกนอกเครื่องในทุกกรณี การตั้งค่าและตัวนับลิมิตเก็บใน chrome.storage.local และลบออกได้ตลอดเวลาจากแท็บตั้งค่า อ่านนโยบายฉบับเต็มได้ที่ หน้านโยบายความเป็นส่วนตัว
⚠️ TikTok มีระบบตรวจจับการใช้งานอัตโนมัติ — โปรดใช้ด้วยความระมัดระวัง ตั้งหน่วงเวลาช้า ๆ และลิมิตต่ำ ๆ การใช้เครื่องมืออัตโนมัติเป็นความรับผิดชอบของผู้ใช้เอง

ส่วนขยายนี้เป็นเครื่องมือของบุคคลที่สาม ไม่ได้เกี่ยวข้องหรือได้รับการรับรองจาก TikTok
