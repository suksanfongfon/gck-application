Lazada Product Finder

เร็ว ๆ นี้ เวอร์ชัน 1.0.0 • สำหรับ Lazada Thailand

ส่วนขยาย Chrome สำหรับนักขายออนไลน์และ affiliate marketer — ค้นหาสินค้า Lazada ตาม keyword กรองตามรีวิว/ยอดขาย/เรตติ้ง แล้ว export เป็น CSV หรือ copy ไป Google Sheets ได้ทันที พร้อมสร้าง affiliate link อัตโนมัติ

⬇ ดาวน์โหลด (เร็ว ๆ นี้) ดูวิธีใช้งาน

ความสามารถ

ออกแบบสำหรับนักขาย affiliate ที่ต้องการหาสินค้าดีรวดเร็ว

🔍

ค้นหาสินค้า

ใส่ keyword แล้วค้นหาได้หลายหน้าพร้อมกัน ระบบเปิด tab Lazada อัตโนมัติและดึงข้อมูลสินค้าทั้งหมดกลับมา

⚙️

กรองตามเงื่อนไข

กำหนดรีวิวขั้นต่ำ, ยอดขายขั้นต่ำ, เรตติ้งขั้นต่ำ เพื่อคัดเฉพาะสินค้าที่น่าเชื่อถือและขายดีจริง

📄

Scrape หน้าที่เปิดอยู่

เปิด Lazada ค้างไว้แล้วกด "Scrape หน้านี้" — ดึงข้อมูลสินค้าจากหน้าที่กำลังเปิดโดยไม่ต้องค้นใหม่

📋

Copy เป็น TSV

คัดลอกผลลัพธ์เป็นรูปแบบ TSV แล้ววางลง Google Sheets ได้ทันที ข้อมูลเข้า column ถูกต้องทุกช่อง

⬇️

Export CSV

ดาวน์โหลดผลลัพธ์เป็นไฟล์ .csv เพื่อนำไปวิเคราะห์ใน Excel หรือนำเข้าระบบอื่น ๆ ได้เลย

🔗

Affiliate Link

แปลง URL สินค้าเป็น affiliate link อัตโนมัติ รองรับหลายรูปแบบทั้ง Lazada Deeplink และ custom parameter

วิธีใช้งาน

3 แท็บ 3 ฟังก์ชัน ใช้งานได้ทันทีหลังติดตั้ง

①

แท็บ "ค้นหา" — หาสินค้า

กรอก keyword เช่น "กระเป๋า" หรือ "โทรศัพท์" ตั้งเงื่อนไขกรอง (รีวิวขั้นต่ำ, ยอดขายขั้นต่ำ, เรตติ้ง, จำนวนหน้า) แล้วกด ค้นหาสินค้า ระบบจะเปิด tab Lazada อัตโนมัติและดึงข้อมูลกลับมา

②

แท็บ "ผลลัพธ์" — ดูและ export

เมื่อค้นเสร็จ ผลลัพธ์จะแสดงชื่อสินค้า ราคา เรตติ้ง รีวิว ยอดขาย และลิงก์ affiliate กด คัดลอก TSV เพื่อวาง Google Sheets หรือ ดาวน์โหลด CSV เก็บไว้ใช้ต่อ

③

แท็บ "ตั้งค่า" — ปรับ affiliate และคอลัมน์

เลือกคอลัมน์ที่ต้องการ export (ชื่อ / ราคา / เรตติ้ง / รีวิว / ยอดขาย / URL) และตั้งค่า affiliate link ตามโปรแกรมที่ใช้

💡 เคล็ดลับ: ถ้าเปิดหน้า Lazada ค้างไว้แล้ว ให้กด "Scrape หน้านี้" แทนการค้นใหม่ — ได้ผลลัพธ์เร็วกว่าและไม่ต้องรอโหลด tab

สิทธิ์การเข้าถึงที่ใช้

  • storage — บันทึกการตั้งค่าและ affiliate config ในเครื่อง
  • activeTab, tabs, scripting — เปิด tab Lazada อัตโนมัติและดึงข้อมูลสินค้า
  • clipboardWrite — คัดลอก TSV ไปยัง clipboard
  • downloads — ดาวน์โหลดไฟล์ CSV
  • เข้าถึง lazada.co.th เท่านั้น
