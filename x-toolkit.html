X Toolkit

พร้อมใช้งาน เวอร์ชัน 1.1.0 • สำหรับ X (Twitter)

เครื่องมือจัดการ X (Twitter) ครบในตัวเดียว — follow / unfollow แบบปรับความถี่ได้ พร้อมโหมดรีโพสต์/ไลก์ตามเกณฑ์ ยกเลิกรีโพสต์ ลบโพสต์ และโพสต์สินค้า Lazada แนบลิงก์ affiliate อัตโนมัติ

＋ เพิ่มลงใน Chrome ดูตัวอย่าง

ความสามารถ

ครอบคลุมหลายโหมดการทำงาน ปรับความถี่และลิมิตได้ทุกโหมด

🧹

โหมด Unfollow

เลิกติดตามเฉพาะคนที่ไม่ติดตามกลับ (non-mutual) พร้อม Whitelist รายชื่อที่ห้าม unfollow

➕

โหมด Follow

กดติดตามตามเงื่อนไข — เฉพาะบัญชี Verified, keyword ใน bio, หรือผู้ติดตามขั้นต่ำ

🔁

โหมด Feed

รีโพสต์ + กดไลก์โพสต์ที่ยอดดูถึงเกณฑ์ กรองเฉพาะวิดีโอ/คำค้น และติดตามเจ้าของที่มีผู้ติดตามถึงเกณฑ์

↩️

โหมด Unrepost

ยกเลิกรีโพสต์ที่ผลงานไม่ดี ตามเกณฑ์คอมเมนต์ / ถูกใจ / ยอดดู เลือกตรรกะ AND/OR ได้

🗑️

โหมด Delete

ลบโพสต์ของเราเองตามเงื่อนไข engagement / keyword / อายุโพสต์ (ลบถาวร — ต้องตั้งเงื่อนไขเสมอ)

🛒

โหมด Lazada

ค้นหาสินค้า Lazada แล้วโพสต์ลง X อัตโนมัติ พร้อมแนบลิงก์ affiliate (Template หรือเรียก API)

⚙️ ทุกโหมดปรับ หน่วงเวลาแบบสุ่ม (min–max), จำนวนสูงสุดต่อรอบ และ ลิมิตต่อชั่วโมง / ต่อวัน ได้ มีตัวนับสดและทำงานต่อแม้ปิด popup รวมถึงรันคนละโหมดในคนละแท็บพร้อมกันได้

ตัวอย่างการใช้งาน

ดูหน้าตาจริงก่อนติดตั้ง

วิธีติดตั้ง

ติดตั้งผ่าน Chrome Web Store ได้ในไม่กี่คลิก ไม่ต้องตั้งค่าอะไรเพิ่ม

1

กด “เพิ่มลงใน Chrome”

คลิกปุ่ม เพิ่มลงใน Chrome บนหน้านี้ หรือค้นหา “X Toolkit” ใน Chrome Web Store โดยตรง

2

ยืนยันการติดตั้ง

กด เพิ่มส่วนขยาย ในหน้าต่างยืนยัน Chrome จะติดตั้งให้อัตโนมัติภายในไม่กี่วินาที

3

ปักหมุดไอคอน

คลิกไอคอนต่อจิ๊กซอว์ 🧩 มุมขวาบน แล้วกดหมุด 📌 ข้างชื่อ X Toolkit เพื่อให้ไอคอนแสดงตลอดเวลา

4

เริ่มใช้งาน

คลิกไอคอนเพื่อเปิด popup เลือกโหมดที่ต้องการ ตั้งค่าความถี่/ลิมิต แล้วกด Start

สิทธิ์การเข้าถึงที่ใช้

  • storage — บันทึกการตั้งค่าและตัวนับในเครื่อง
  • activeTab, tabs, scripting — ทำงานกับแท็บ X ที่เปิดอยู่ และเปิดโปรไฟล์ในแท็บซ่อนเพื่อเช็คผู้ติดตาม
  • เข้าถึง x.com / twitter.com และ Lazada API (สำหรับโหมด Lazada)
⚠️ ข้อควรระวัง: ใช้กับ บัญชีของคุณเอง เท่านั้น X มีลิมิตการ follow/unfollow/โพสต์ต่อวัน — ตั้งความถี่ช้า ๆ และลิมิตต่อชั่วโมง/วันเสมอ เพื่อลดความเสี่ยงโดนจำกัดบัญชี การรันหลายโหมดพร้อมกันจะเสี่ยงโดนตรวจจับเป็นบอทสูงขึ้น
